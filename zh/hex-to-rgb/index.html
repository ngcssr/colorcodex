Hex 转 RGB

Hex 色号和 RGB 数值描述的是同一组红、绿、蓝通道。Hex 用十六进制对保存每个通道，RGB 则用 0 到 255 的十进制数字表示。

如何将 Hex 转为 RGB

输入三位或六位 Hex 色号后，转换器会展开简写值，计算 RGB 通道，并显示匹配的 HSL、HSV、OKLCH 和 CMYK 值。

在 CSS 中使用 RGB 值

RGB 值适合查看或编辑屏幕颜色的红、绿、蓝组成。需要可直接粘贴的 CSS 颜色函数时，可使用 HTML/CSS 列。
